1. .envファイルの配置
<ユーザ名>から.envファイルをもらい、.envファイルをルートディレクトリ直下に配置
2. プロジェクトの作成
  ```
  docker-compose run
  ```
3. コンテナの起動
  ```
  docker-compose up -d
  ```
4. コンテナの停止
  ```
  docker-compose down
  ```
5. 動作確認
http:localhost:80にアクセスできるか確認
アクセスできたら成功
